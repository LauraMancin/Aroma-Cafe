🐱☕ Cat Café & Adoção

Plataforma web que une um café temático com gatos e um sistema de adoção responsável, permitindo também doações para ajudar no cuidado dos animais.

🎯 Sobre o Projeto

O Cat Café & Adoção é um sistema desenvolvido com o objetivo de proporcionar uma experiência interativa para os usuários, permitindo:

- Conhecer gatos disponíveis para adoção 🐾
- Contribuir com doações 💖
- Explorar um cardápio temático inspirado em gatos ☕
- Incentivar a adoção responsável

Além disso, o projeto tem um impacto social ao ajudar animais em situação de vulnerabilidade.

🚀 Funcionalidades
- 🐱 Catálogo de gatos com fotos, descrição e personalidade
- 💖 Sistema de doações para apoio ao café/abrigo
- ☕ Cardápio temático com bebidas inspiradas em gatos
- 📅 Agendamento de visitas ao espaço (opcional)
- 👤 Sistema de usuários (login e cadastro) (se você for implementar)
