# ☕🐱 Aroma Café — Sistema Web para Gato Café

Este repositório contém o desenvolvimento do **Aroma Café**, projeto criado como **Trabalho de Conclusão de Curso (TCC)** do curso de **Desenvolvimento de Sistemas**.
O **Aroma Café** é um conceito de *Gato Café*: um espaço onde os clientes podem desfrutar de café e outras bebidas enquanto interagem com gatos em um ambiente acolhedor e controlado. O objetivo do sistema é oferecer uma plataforma digital que facilite a gestão do estabelecimento e a experiência dos usuários.

# 🎯 Objetivo do Projeto

Desenvolver um **site/sistema web** que permita apresentar o café ao público e oferecer funcionalidades que auxiliem tanto os clientes quanto a administração do estabelecimento.

Entre os principais objetivos estão:

* Apresentar o **Aroma Café** e sua proposta;
* Mostrar os **gatos disponíveis para interação ou adoção**;
* Disponibilizar **informações sobre cardápio e ambiente**;
* Criar uma interface intuitiva e agradável para os usuários;
* Aplicar na prática os conhecimentos adquiridos durante o curso de **Desenvolvimento de Sistemas**.

# 🧩 Funcionalidades previstas

* Página inicial com apresentação do café;
* Página sobre o espaço e a proposta do Gato Café;
* Catálogo dos gatos presentes no local e disponiveis para adoção;
* Informações sobre o cardápio;
* Área administrativa para gerenciamento de informações;
* Interface responsiva para diferentes dispositivos;
